# TradFi × Onchain

傳統金融點樣接入鏈上市場 — for HK finance professionals.

## What is this

A content site explaining how traditional finance (TradFi) is integrating with onchain products. Built for Hong Kong finance professionals — IBD, S&T, prop trading, asset management, family offices — who want to understand what's happening and how to participate.

## Content

- **Market Landscape** — BlackRock BUIDL, JP Morgan Kinexys, Goldman GS DAP, HK govt tokenized green bonds, HSBC Orion, and more
- **Polymarket** — Prediction market. Buy outcomes, not assets. $9.7B 30-day volume. ICE (NYSE parent) invested $600M.
- **Hyperliquid** — Perp DEX + HIP-3 pre-IPO stock perps (SpaceX, OpenAI, Anthropic). $5-10B daily volume.
- **Onchain AMM** — Permissionless markets. Trading, LP (liquidity provision), and launchpads (PumpFun, Raydium LaunchLab).
- **HK Regulation** — SFC VATP licensing, staking rules, stablecoin ordinance, family office tax concessions.

## Tech Stack

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Vercel](https://vercel.com) — hosting

## Development

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output in dist/
npm run preview  # preview production build
```

## Deploy

Connected to Vercel. Push to `main` → auto deploy.

Custom domain: `tradfi.onchain.hk`
